Les variables var, let et const

  • Var : sert à définir des variables globales. Aujourd’hui c’est plutôt maladroit pour plusieurs raisons
  • Let : Sert à définir des variables qui ont un résoluteur de portée niveau block (ou block scope). Ces variables sont modifiables.
  • Const : Sert à définir des variables constantes qui ont aussi un résoluteur de portée niveau block. Ces variables sont constantes donc non modifiables.

Pour choisir il suffit de se demander si je voudrais que la valeur change. Si je crée une variable qui pointe vers une <div> par exemple, elle ne changera pas.

Déclarer une variable et lui affecter une valeur

Pour déclarer une variable, on utilise le type de variable puis le nom de notre variable.

let maVariableLet;

Pour affecter une variable, on utilise le nom de la variable puis on utilise l'opérateur d'affectation = pour lui associer une valeur.

maVariable = 1

Dans le cas des const, il faut directement déclarer et associer une valeur, sinon on aura automatiquement une erreur

const maVariableConst = "Une valeur"

Si on oublie de déclarer une variable avant d’y affecter une valeur sans “use script” alors il recevra un var comme type. C’est une mauvaise pratique !
