Les variables var, let et const

  • Var : sert à définir des variables globales. Aujourd’hui c’est plutôt maladroit pour plusieurs raisons
  • Let : Sert à définir des variables qui ont un résoluteur de portée niveau block (ou block scope). Ces variables sont modifiables.
  • Const : Sert à définir des variables constantes qui ont aussi un résoluteur de portée niveau block. Ces variables sont constantes donc non modifiables.

Pour choisir il suffit de se demander si je voudrais que la valeur change. Si je crée une variable qui pointe vers une <div> par exemple, elle ne changera pas.
